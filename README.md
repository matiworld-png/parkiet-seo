# parkiet-seo

Statyczny hosting dla `parkiet-deindex-urls.xml` — sitemap przyspieszający
deindeksację martwych URL-i (HTTP 404) sklepu parkiet.pl w Google.

Plik referencjowany z `robots.txt` parkiet.pl jako cross-host sitemap
(oficjalna funkcja Google Search Central).

Zawiera 609 URL-i z `<lastmod>=2026-05-19`. Pliki nie zawierają danych
osobowych ani wrażliwych — tylko publiczne ścieżki URL z parkiet.pl.
